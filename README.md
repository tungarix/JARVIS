# ⬡ JARVIS
### Just A Rather Very Intelligent System

A personal AI assistant that runs completely locally on your computer.
No API keys, no internet required for AI — fully private.

---

## 🚀 Features

- 💬 Chat interface with conversation history
- 🧠 Local AI powered by Ollama (qwen2.5:3b)
- 💾 Memory system — remembers important info about you
- 🖥️ 40+ computer commands (open apps, folders, websites)
- 🖱️ Mouse & keyboard control
- 📁 File & folder creation/deletion
- ⏰ Task scheduler & timer
- 🌐 Web shortcuts (YouTube, Google, Instagram...)
- 📝 Note taking system
- 📊 Real-time CPU & RAM monitoring

---

## 🛠️ Requirements

- Python 3.10+
- [Ollama](https://ollama.com) 
- qwen2.5:3b model

---

## ⚡ Installation
```bash
# 1. Install Ollama and pull the model
ollama pull qwen2.5:3b

# 2. Install Python dependencies
pip install ollama customtkinter psutil pyautogui keyboard schedule

# 3. Run JARVIS
python jarvis_ui.py
```
